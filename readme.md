# Handyman
an app made for the second semester project

------------
### technologies used:
- React Native
- supabase

### Run the project
first create a .env file containing SUPABASE_KEY & SUPABASE_URL.
then run
```
npm install
expo start
```
for more information on [expo](https://docs.expo.dev/get-started/installation/).
